# LLM Engineering Handbook

[![Open site](https://img.shields.io/badge/open-site-brightgreen?logo=githubpages&logoColor=white)](https://einouqo.github.io/llm-handbook/)
[![Open PDF](https://img.shields.io/badge/open-pdf-red?logo=adobeacrobatreader&logoColor=white)](https://einouqo.github.io/llm-handbook/llm-handbook.pdf)

This handbook collects practical notes about LLMs, inference, deployment, and model variants into a single Typst-based reference. It is meant to be a compact, evolving guide rather than a formal textbook.

The content was crafted from many notes collected over a long time, so it still lacks references. Any corrections, improvements, and especially new references are welcome.
